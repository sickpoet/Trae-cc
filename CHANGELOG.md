# 📋 更新日志

## [1.0.0] - 2025-03-15

### ✨ 新增功能

- 🚀 快速注册功能 - 自动生成临时邮箱并自动完成注册流程
- 🎨 全新 UI 界面 - 重构整个界面设计，更加美观易用
- 📊 账号使用量实时监控
- 🔄 一键切换账号功能
- 📥 账号导入/导出功能
- 🌙 深色/浅色主题切换

### 🔧 优化改进

- 优化页面加载速度
- 改进账号管理逻辑
- 优化错误提示信息
- 改进表单验证机制

### 🐛 Bug 修复

- 修复快速注册时验证码获取问题
- 修复 Token 拦截失败的问题
- 修复邮箱验证码解析问题
- 修复若干界面显示问题

## [2.0.0] - 2025-02-XX

### ✨ 新增功能

- 基于 Tauri 2.0 重构项目架构
- 新增 Rust 后端处理
- 新增多账号管理功能
- 新增机器码管理功能

### 🔧 优化改进

- 从 Electron 迁移到 Tauri，大幅减小安装包体积
- 提升应用启动速度
- 优化内存占用
